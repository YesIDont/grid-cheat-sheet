Areas and grid-lines names

Area combines grid-row and grid-column lines props into one:


            grid-area: row-start column-start row-end column-end
        


          grid-template-columns:
            [grid-start] 1fr
            [content-start] 1fr
            [content-end] 1fr
            [grid-end];
          grid-template-rows:
            [header-start] 1fr
            /* each line can have added more than one name */
            [header-end body-start] 1fr
            [body-end footer-start] 1fr
            [footer-end];
1: grid-column: grid-start / grid-end
2: grid-area: body-start / grid-start / footer-end / content-end
3: Grid item
4: grid-column: content-end / grid-end

Auto (magick) content areas

By using name template [name-start] and [name-end] you can define rows and columns lines and use just the [name] key word for area.


          grid-template-columns:
            [content-start]
            1fr 1fr
            [content-end]
            1fr;
          grid-template-rows:
            1fr
            [content-start]
            5fr 1fr
            [content-end];
        


          [selector] {
            grid-area: content;
          }
1: grid-column: 1 / -1
2: grid-area: content
3: Grid item
4: grid-column: 3 / -1
